# FideCompro - Avance 2

Proyecto desarrollado para el curso Programación Cliente-Servidor Concurrente de la Universidad Fidélitas.

## Descripción

FideCompro es una aplicación de escritorio desarrollada en Java Swing para administrar usuarios, clientes y productos de una empresa de venta al por mayor.

Este segundo avance implementa seis de las doce historias de usuario definidas, equivalentes al 50% del proyecto.

## Funcionalidades implementadas

- Inicio de sesión mediante usuario y contraseña.
- Registro de usuarios.
- Control de usuarios duplicados.
- Registro de clientes.
- Búsqueda de clientes por cédula.
- Control de clientes duplicados.
- Registro de productos.
- Búsqueda de productos por código.
- Control de productos duplicados.
- Validación de datos numéricos.
- Visualización de usuarios, clientes y productos mediante tablas.

## Conceptos utilizados

- Clases y objetos.
- Encapsulamiento.
- Herencia.
- Polimorfismo.
- Colecciones `ArrayList`.
- Excepciones personalizadas.
- Interfaz gráfica con Java Swing.
